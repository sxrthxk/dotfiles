# Collections of my dotfiles for
- i3
- rofi
- polybar
- zshrc

Would be adding more later.

### Dependencies:
 - i3-wm
 - polybar
 - rofi
 - Emacs (Especially for Org mode and Lisp programming)
 - zsh (oh-my-zsh with powerlevel10k theme)
 - LunarVim as default text editor / still learning NvChad

Also planning to add picom, dunst and alacritty.

## Install

```bash
git clone https://github.com/sxrthxk/dotfiles.git ~/
```

Note: Run `./setup.sh` to automatically symlink the files/folders.
